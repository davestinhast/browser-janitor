# Browser Janitor

Browser Janitor is a safe browser cleanup tool for people who want a faster,
lighter browser profile without losing accounts, history, passwords, sessions or
bookmarks.

It scans Chrome, Edge, Brave, Chromium and Firefox profiles, reports how much
space is used by regenerable cache folders, and only cleans targets that are
explicitly marked as safe.

## Features

- console-first interface that works well as a Windows `.exe`
- safe cache scanner with dry-run cleanup
- HTML and Markdown reports
- browser extension permission audit
- conservative guards for sensitive profile files
- no runtime dependencies

## What it cleans

- HTTP cache
- JavaScript and WASM code cache
- GPU and shader caches
- WebGPU Dawn cache
- browser metrics logs
- crash reports
- Firefox startup/cache folders
- service worker cache storage

These files are regenerated by the browser. After cleaning, the first load of
some sites may be slower while the browser rebuilds cache.

## What it never cleans by default

- browsing history
- cookies
- saved passwords
- bookmarks
- autofill and payment data
- browser login databases
- active session restore data
- extension settings

The cleaner also has a protected-name guard. If a future cleanup rule accidentally
points at a sensitive file such as `History`, `Cookies`, `Login Data`,
`Bookmarks`, `places.sqlite`, `logins.json` or session data, Browser Janitor
blocks it.

## Usage

Scan only:

```powershell
python -m browser_janitor scan
```

Write a report:

```powershell
python -m browser_janitor scan --report browser-report.md
```

Dry-run cleaning:

```powershell
python -m browser_janitor clean
```

Actually clean safe cache targets:

```powershell
python -m browser_janitor clean --apply
```

JSON output:

```powershell
python -m browser_janitor scan --json
```

Run a full health check:

```powershell
python -m browser_janitor doctor --report browser-report.html
```

Audit extensions:

```powershell
python -m browser_janitor extensions
```

## Windows executable

Build a console executable with PyInstaller:

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\build-exe.ps1
```

The output is:

```text
dist\BrowserJanitor.exe
```

It is built with the console subsystem, so double-clicking it opens like a CMD
program instead of a graphical app.

## Safety model

Browser Janitor is intentionally conservative:

1. It only scans known browser profile locations.
2. It only cleans known cache directories.
3. `clean` is a dry run unless `--apply` is passed.
4. Sensitive profile databases are blocked by name even if they appear under a
   configured cleanup path.
5. Locked files are skipped instead of forcing browser processes to close.

For best results, close your browsers before running `clean --apply`.
